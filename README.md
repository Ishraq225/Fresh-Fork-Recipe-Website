# 🍽️ FreshFork — Healthy Recipe Website

## 🌱 Overview

**FreshFork** is a modern recipe website designed for diet-conscious individuals who want to prepare **healthy, balanced meals** with ease. The platform provides carefully curated recipes for **breakfast, lunch, and dinner**, each with **accurate ingredient measurements** and **calorie tracking** to help users maintain a healthy lifestyle.

Whether you're trying to lose weight, maintain your fitness, or simply eat clean, FreshFork makes healthy cooking simple and accessible.

---

## ✨ Features

### 🥗 1. Healthy Meal Categories

* **Breakfast, Lunch, Dinner** recipes
* Designed specifically for diet-conscious users
* Focus on balanced nutrition and portion control

### ⚖️ 2. Accurate Measurements

* Step-by-step recipes with **precise ingredient quantities**
* Helps users cook consistently and avoid over/under consumption

### 🔥 3. Calorie Tracking

* Each recipe includes **calorie information**
* Helps users monitor daily intake and maintain diet goals

### 📩 4. Contact Form (Feedback & Complaints)

* Users can:

  * Submit feedback
  * Report issues
  * Share suggestions
* Messages are sent directly to the website author's **email**

### 🌐 5. Social Media Integration (Footer)

Users can connect and explore more content through:

* 📸 Instagram
* 📘 Facebook
* ▶️ YouTube (watch recipe videos and tutorials)

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Styling:** Custom CSS / Responsive Design
* **Form Handling:** Email integration

## 🚀 How It Works

1. Browse recipes by meal type (Breakfast, Lunch, Dinner)
2. Select a recipe
3. View:
   * Ingredients with exact measurements
   * Step-by-step instructions
   * Total calorie count
4. Prepare your meal with confidence
5. Share feedback or suggestions via the contact form
